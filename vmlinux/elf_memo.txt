    uImage      //U-bootが使うimage file。mkimageで、zImageの先頭に0x40サイズ分のheaderを追加したfileである。
    bootpImage  //
    xipImage    //
    bzImage     //big zImage...使ってことがない、不明
    ...

vmlinux:
--------------------------------------------------------------------------------
    vmlinuxは、起動可能、非圧縮、および圧縮可能なKernel Imageである。
    vmは仮想メモリの略、これは、Linuxが仮想メモリをサポートしていることを意味する。
    したがって、vmという名前だ。

    基本的に、ELF(Executable and Linkable Format)形式のファイルである。


--------------------------------------------------------------------------------
ELF (Executable and Linkable Format)
--------------------------------------------------------------------------------

     Linking View                             Execution View

     +--------------------------------+       +------------------------------------+
     | ELF Header                     |       |     ELF Header                     |
     +--------------------------------+       +------------------------------------+
     | Program Header Table(Optional) |       |     Program Header Table           |
     +--------------------------------+       +------------------------------------+
     | Section 1                      |       |                                    |
     +--------------------------------+       |     Section 1(.text)               |
     |                                |       +------------------------------------+
     | Section 2                      |       |                                    |
     +--------------------------------+       |     Section 2(.rodata)             |
     |                                |       +------------------------------------+
     | ...                            |       |     Section 3(.data)               |
     +--------------------------------+       |                                    |
     |                                |       +------------------------------------+
     | Section n                      |       |                                    |
     |                                |       |     ...                            |
     +--------------------------------+       +------------------------------------+
     | Section Header Table           |       |     Section Header Table(Optional) |
     +--------------------------------+       +------------------------------------+


カーネルのマクロにより、Sectionプロパティーが定義されて、優先度の高いソースコードは、前に置かれる。

マクロ:
    arch_initcall
    arch_initcall_sync
など
